peasy-tools-embed

Embed PeasyTools widgets on any website. Zero dependencies, Shadow DOM isolation, 3 themes.

npm install peasy-tools-embed Visit peasytools.com

Quick Start

<!-- Place widget where you want it -->
<div data-peasy-tools="tool" data-slug="compress" data-theme="light"></div>

<!-- Load script once -->
<script src="https://cdn.jsdelivr.net/npm/peasy-tools-embed@1/dist/embed.min.js"></script>
